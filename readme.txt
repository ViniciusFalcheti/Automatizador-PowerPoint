Segunda versão do Programa de automação de criação de slides de palavra do culto online, feito para auxiliar nos cultos da Comunidade Cristã de Ribeirão Preto. Nessa versão é utlizado um arquivo de entrada.docx para a criação de um arquivo PowerPoint.

- Utilizada linguagem Python no desenvolvimento
 	-  Utilizado biblioteca pptx para a modificação do arquivo do Powerpoint
 	-  Utilizado biblioteca Tkinter para a criação da interface gráfica de escolha de tema
 	-  Utilizado docx para a leitura do arquivo word de base
